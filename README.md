# crud-php

1. Execute as queries em init.sql (database/init.sql) em sua database
2. Verifique e acerte o db_config.php (database/db_config.php) para configurações de acesso ao banco de dados

Projeto feito em:
  - PHP 7.4.3
  - MySQL 8.0.27
  
Testado em:
  - Linux Ubuntu 20.04
  - Mozilla Firefox 93.0
